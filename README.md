# Graduation-Project

1. Check Seat
 - 학생이 자리에 있는지 없는지 확인
 - 과정 01 : 학생이 고개를 갸웃거리고 있을 때 인식이 잘 안되는 부분을 보완하기 위해 캡쳐한 프레임을 왼쪽으로 90도, 오른쪽으로 90도까지 10도씩 사진을 회전시켜 얼굴을 인식
 - 과정 02 : 얼굴 인식이 될 경우 학생이 자리에 앉아있는 것으로 판단하여 다음 알고리즘으로 넘어간다. 
 - 과정 03 : 얼굴 인식이 안될 경우 학생이 자리에 앉아있지 않은 것으로 판단
